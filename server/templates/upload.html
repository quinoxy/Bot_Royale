📋 How to Write Your Bot

Your bot communicates via stdin/stdout. On each turn:

  • Read one line from stdin → JSON with the board state
  • Print one line to stdout → row col (your move)

Board state JSON format:

{
  "board": [[count, owner], ...],  // 7×7 grid
  "player": 1,                     // 1=RED, 2=BLUE (you)
  "move_number": 5,
  "rows": 7, "cols": 7
}
// owner: 0=NONE, 1=RED, 2=BLUE

Example bot (plays first valid cell):

import json, sys

while True:
    line = input()
    state = json.loads(line)
    me = state["player"]
    board = state["board"]

    for r in range(7):
        for c in range(7):
            owner = board[r][c][1]
            if owner == 0 or owner == me:
                print(f"{r} {c}", flush=True)
                break
        else:
            continue
        break

Rules:

  • ⏱ You have 2 seconds per move
  • ❌ Invalid move = disqualification
  • 💥 Crash or timeout = disqualification
  • 🏆 Win by eliminating all opponent pieces
