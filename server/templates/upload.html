📋 How to Write Your Bot

Your bot is a Python script that communicates via stdin/stdout in a loop. On each turn:

  • Read one line from stdin → JSON with the board state & time info
  • Print one line to stdout → row col (your move)

Board state JSON format (received each turn):

{
  "board": [[count, owner], ...],  // 7×7 grid
  "player": 1,                     // 1=RED, 2=BLUE (you)
  "move_number": 5,
  "rows": 7, "cols": 7,
  "my_time": 58.3,                 // your remaining time (seconds)
  "opp_time": 55.1                 // opponent's remaining time
}
// owner: 0=EMPTY, 1=RED, 2=BLUE

Example bot (plays first valid cell):

import json

while True:
    state = json.loads(input())
    me = state["player"]
    board = state["board"]

    for r in range(state["rows"]):
        for c in range(state["cols"]):
            owner = board[r][c][1]
            if owner == 0 or owner == me:
                print(f"{r} {c}", flush=True)
                break
        else:
            continue
        break

Rules:

  • ⏱ Each player has a 60-second time bank for the entire game (no per-move limit — spend your time wisely)
  • ❌ Invalid move or bot crash → opponent wins by default
  • 💥 Running out of time → opponent wins by default
  • 🏆 Win by eliminating all opponent pieces via chain reactions
